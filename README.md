## What does it use?
.NET Framework 4.6 and C# for the backend. WPF/XAML for the interface.

## Supported platforms
At this time, only Windows 7+ is supported (Windows 7, 8, 8.1, 10 | 32/64 Bits).

## Can I help you?
Of course yes! Any pull-request will be considered.
